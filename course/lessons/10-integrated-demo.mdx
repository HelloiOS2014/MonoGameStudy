## Goal

Connect the previous concepts into the playable collector demo capstone.

## Run

Manual run:

```bash
dotnet run --project demo/integrated-demo/IntegratedDemo.csproj
```

Automated smoke:

```bash
env DEMO_SMOKE_EXIT_AFTER_FRAMES=160 dotnet run --project demo/integrated-demo/IntegratedDemo.csproj --no-restore
```

## Observe

Manual mode opens the demo. Press `Enter`, collect the yellow pickups, avoid red hazards, and press `Enter` or `R` after win/loss to restart.

Smoke output includes:

```text
Phase: started.
Collect: score=3/3.
Phase: won.
Phase: restarted.
Smoke: exit.
```

## Expected Visual State

Collector arena starts at Start phase, smoke collects all pickups, reaches Won, restarts, and exits.

## Walkthrough

The demo has four phases: `Start`, `Playing`, `Won`, and `Lost`. `DemoState` is the deterministic model. It owns player position, score, pickups, hazards, phase transitions, circle collision checks, and restart behavior.

`DemoTypes.cs` defines the phase and input/result records. `DemoState.CreateDefault` starts in `Start`; a `DemoInput` with `StartPressed` calls `ResetRun`, enters `Playing`, clears score, and restores pickups. Moving right through the deterministic pickup layout raises the score to `DemoState.TargetScore`, which sets `Phase` to `Won`.

`Game1` connects the model to MonoGame by polling input, loading content, drawing the arena, logging phase changes, and playing the collect sound. Smoke mode disables vsync, uses variable timestep at the MonoGame layer, then feeds the model a fixed `1/60` simulation step from `DemoSmokeSettings` so the capstone can be checked without manual input.

This integrated demo is the course capstone and validation harness. It combines content, input, audio, collision, state, and smoke verification from earlier lessons; it is not a production-game expansion target.

## Common Failures

- GUI smoke can hang under vsync/fixed timestep on macOS; this demo disables vsync and uses fixed smoke simulation time.
- The circles are drawn as filled rectangles in this learning demo; collision still uses circle math in `DemoState`.
- If content fails to load, rerun restore/build without `--no-restore` once and inspect `Content/Content.mgcb`.
